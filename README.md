# Spring-Data-Jpa

Study


## Entity

Entity는 기본적으로 클래스 이름 사용한다. 


### Entity 상태

- Transient : JPA가 관리하지 않는 상태

- Persistent : JPA가 관리중인 상태

- Detached : JPA가 더이상 관리하지 않는 상태 

- Removed : JPA가 관리하긴 하지만, 삭제하기로 한 상태


: Cascade : 엔티티의 상태를 전파 시키는 옵션
